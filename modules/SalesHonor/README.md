# Sales Honor Roll Module

This module provides a dedicated page to recognize and celebrate the achievements of top-performing sales agents.

## Features

-   **Podium View**: A prominent display for the top 3 sales members, styled with gold, silver, and bronze themes.
-   **Leaderboard**: A detailed table ranking other top performers.
-   **Key Metrics**: Displays important sales data like total sales volume, policies sold, and percentage of quota achieved.
-   **Visual Appeal**: Designed with a premium and celebratory aesthetic to properly honor achievements.
