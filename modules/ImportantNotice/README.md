# Important Notices Module

This module provides a dedicated page to display critical announcements, policy updates, and other important information to all users. It supports multiple layouts to best suit user preference.

## Features

-   **Multiple Views**: Switch between a List, Grid, or Timeline layout.
-   **Rich Content**: Notices can now include images and file attachments.
-   **Interactive**: Expand notices to view full details and download attached files.
-   **Priority-Based Display**: Notices are visually distinguished based on their priority level (Critical, Update, Info) using colors and icons.
-   **Centralized Communication**: Acts as a single source of truth for important, non-transient communications.
