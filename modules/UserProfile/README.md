# User Profile Module

This module displays the current user's profile information and provides access to account settings.

## Features

-   **User Summary**: Displays the user's avatar, name, and role.
-   **Navigation**: Allows switching between different views like "Information" and "Settings".
-   **Information View**: Shows detailed user attributes such as email, department, etc.
-   **Settings View**: A placeholder for future user-configurable settings.
